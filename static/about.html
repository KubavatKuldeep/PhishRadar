About This Project

PhishRadar is a full-stack cybersecurity web application developed as a 2-week internship project. It provides a unified platform for detecting phishing threats across multiple attack vectors — URLs, Emails, and QR Codes.

The system uses a rule-based detection engine that checks for over 11 threat indicators including brand impersonation, URL authority spoofing, typosquatting, suspicious TLDs, and more. Google Gemini AI is used exclusively to explain the findings in plain language — the AI never makes the security verdict.

The platform follows a privacy-first design — no user data is stored, no database is required, and all analysis is performed in-memory.

Key Features

  • URL, Email & QR Code phishing detection
  • Real redirect chain tracing & visualization
  • 11 detection rules including typosquatting & authority spoofing
  • AI-powered threat explanations via Gemini
  • Individual finding severity with weight scoring
  • Privacy-first — no database, no stored data
